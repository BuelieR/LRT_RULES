# What's this project?
**This is a document website about LRT rules in China that was built by a python package which named `mkdocs`.You can also use this project to create your own document website quickly.We agree all people use this project for non-commercial purposes.**

# How to use this project?
**You can only download the file `mkdocs.yml` and the folder `docs` to use this project.You only need to add new files in the folder `docs` to create new pages in your website(Remember edit the file `mkdocs.yml` and make sure install `mkdocs` package).Then run the command `mkdocs serve` in the root directory of the project to start a local server. You can access the website at `http://localhost:8000` in your browser.**

```bash
# Install mkdocs package
pip install mkdocs

# Start a local server
mkdocs serve
```
