# 铁路修建标准

## 车辆编组标准
| 类型 | 编组 | 备注 |
| -------- | -------- | :--------: |
| 地铁列车 | ≧4 | 特殊编组除外，轻铁 轻轨和有轨电车除外 |
| 高铁列车 | ≧8 | - |
