# 铁路修建标准

## 车辆编组标准
| 类型 | 编组 | 备注 |
| -------- | -------- | :--------: |
| 地铁列车 | ≧4 | 特殊编组除外，轻铁 轻轨和有轨电车除外 |
| 高铁列车 | ≧8 | - |

## 站台标准

| 类型 | 最少站台数 | 备注 |
| -------- | -------- | :--------: |
| 地铁站台 | ≧1 | - |
| 高铁站台 | ≧2 | - |
| 混合站台 | ≧3 | 指含2种及以上交通类型的站台 |
| 货运站台 | ≧3 | - |
| 列车停放站台 | ≧1 | 指用于停放列车的站台 |
| 调度站台 | ≧1 | 指用于列车调度的站台 |
| 车库 | ≧1 | 指用于停放列车的地方 |
| 其他交通类型站台 | - | 视情况而定 |
