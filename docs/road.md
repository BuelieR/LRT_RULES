# 公路修建标准

## 定义
**在LRT服务器中，`公路`是指用于连接各地交通流量的公共设施通道。不包含私有交通通道、封闭场合通道等。**

## 公路通行方向
| 类型 | 是否允许单向通行 |
| -------- | -------- |
| 高速公路 | 否 |
| 普通公路 | 是 |
| 其他公路 | 视情况而定 |

## 最少车道
* **车道指道路上单侧车道**

| 类型 | 最少车道数 | 备注 |
| -------- | -------- | -------- |
| 高速公路 | 2车道 | - |
| 普通公路 | 1车道 | - |
| 单侧通行道路 | - | 总共1车道 | - |
| 其他公路 | 视情况而定 | - |
