# 欢迎来到LRT服务器
**LRT服务器是一个Minecraft MTR模组服务器。加入服务器，您需知晓以下事项：**

> * **服务器版本：Fabric 1.20.1**
> * **MTR模组版本：4.0.0**
> * **服务器IP：请自行从QQ群公告获取**
> * **! 游玩LRT服务器，即代表您同意遵守服务器规则。**

# 服务器基本规则
* **禁止在服务器内进行任何违法犯罪行为，违者移交执法、司法机关处理后，视情节轻重服务器管理员予以部分处罚。**
* **禁止炸服、恶意破坏服务器、恶意跑图等行为，违者`拉黑15天`起步，视情况予以延长，情节性质极恶劣者，予以`永久封禁`处理**
* **禁止在服务器内进行任何形式的广告、推广、销售等行为，违者`拉黑15天`起步**
* **因服务器购有部分追加包授权，请谨慎使用追加，禁止以任何形式盗包、向外传包，违者`踢出服务器成员列表`**
* **禁止在服务器内对其他玩家进行任何形式的攻击、伤害、破坏等行为，违者视情况决定处罚，情节严重者`踢出服务器成员列表`**
